# StockT
> 주식 자동매매 프로그램

## About
> 파이썬을 이용하여 투자 알고리즘을 구현하고 트레이딩 시스템을 구축

## Development Environment
- Windows 64bit
- 키움증권 Open API+
- Python 3.6
- PyQt5

## Version
> v0.4.1

## Plan
- 급등주 포착 알고리즘 수정
- 윈도우 작업 스케줄러로 자동 실행 구현
- 당기 순이익 또는 재무제표 참고한 알고리즘 구현
- 매도 구현 (보유한 종목에 대해서만 체크)
- 딥러닝 적용
